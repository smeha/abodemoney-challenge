# Abodemoney Challenge
Abodemoney Challenge

## Tech Stack
- NodeJS (v24.13)
- Node Package Manager (npm v11.8)
- Vite (v7.3)
- React (v19.2)

## How to run locally
```bash
npm install
npm run dev
```

## Built-in lint test
```bash
npm run lint
```

## Run test cases
- watch mode
```bash
npm test
```

- single run (CI-style)
```bash
npm run test:run
```
